Sobre las búsquedas

El sistema tiene un componente de busqueda colocado en diferentes partes del sitio. Este componente es un podersoso buscador al estilo Google que normalmente debería encontrar lo que el usuario esta buscando. Por favor, tome nota de las siguientes reglas para su uso:

  • Normalmente todo lo que usted debe hacer es colocar el criterio de búsqueda en la caja de texto.
  • Para una búsqueda mas específica puede utilizar el formulario de búsqueda avanzada (si está disponible)
  • O colocar el criterio de búsqueda para las columnas segun lo necesario con la ayuda específica
