The KPI Tool is an intuitive and powerful system for managing KPIs for an organization and is unique in that it handles most of the complexity behind KPI management, doing the right thing when appropriate without sacrificing fun.

Esta aplicacion es un proyecto de Codigo Libre patrocinado inicialmente por Artexacta S.R.L. y disponible para todo público. El código se encuentra disponible en https://github.com/Artexacta/KPITool
